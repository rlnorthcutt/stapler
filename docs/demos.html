The Showroom

Three modes. Three documents.

Each demo is a live rendering — no screenshots, no iframes. The library runs in your browser and lays out the document in real time.

Demo 1 · Explicit Mode

The Staple

A 120-Year Journey to Perfection

Three hard-clipped pages with explicit layout control. Each <s-page> is a fixed container — content that overflows is clipped by design.

  • Cover page with no header or footer
  • Running header and footer on pages 2–3
  • Page numbers in n of total format
View Demo

Demo 2 · Flow Mode

The Papyrus Protocol

A History of Binding

A multi-section whitepaper where <page-spacer> elements ensure each chapter starts on a new page. Headers and footers are stamped automatically.

  • Five chapters, each forced to its own page
  • Header and footer on every page
  • Page numbers in running header
View Demo

Demo 3 · Pure Flow

Flatland

A Romance of Many Dimensions

Long-form prose with zero <page-spacer> elements. The library measures every block, computes all page boundaries, and injects spacers automatically.

  • No manual page breaks whatsoever
  • All spacers injected automatically at runtime
  • Running header with total page count
